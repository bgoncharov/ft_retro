# How to use?
First of all, download/clone this repo:
```
https://github.com/bgoncharov/ft_retro.git
```

Get into it and build the library:
```
cd ft_retro
make
```

Now test it with:
```
./ft_retro
```

::: warning
*Enjoy :)*
:::
